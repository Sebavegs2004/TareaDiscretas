para ejecutar el programa tiene que usar gcc en el directorio de la siguente forma:
gcc main.c -o a

para en la siguente linea colocar:

Windows:
	a.exe
Linux:
	./a

para poder leer los grafos en el programa estos deben estar en la carpeta

Por favor seguir las instrucciones ya que de otra forma el codigo no se podra ejecutar de manera efectiva
